# Student Marks Prediction Using Machine Learning

## Project Overview

This project predicts a student's marks based on the number of study hours using the Linear Regression algorithm.

The model learns from historical student data and predicts marks for new study-hour inputs.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## Machine Learning Algorithm

- Linear Regression
- Supervised Learning

---

## Features

- Load dataset from CSV
- Train Machine Learning model
- Predict student marks
- Calculate model accuracy
- Display evaluation metrics
- Generate regression graph

---

## Dataset

Input:
- Study Hours

Output:
- Student Marks

---

## How to Run

```bash
pip install -r requirements.txt
python marks_prediction.py
```

---

## Sample Output

```
Model Accuracy : 0.98

Enter Study Hours : 7

Predicted Marks : 71.62
```
